# Notices

## Rhizomatic provenance

This package is inspired by Myk Bilokonsky's Rhizomatic work:

- https://github.com/mbilokonsky/rhizomatic

Myk's repository contains the Rhizomatic specification, TypeScript and Rust witnesses, conformance vectors, and Chorus agent-memory application.

`pi-rhizomatic` is an independently written Pi/client adapter and network-canary harness. It does not vendor Myk's source code, specification text, conformance vectors, or Chorus implementation.

Future compatibility work should treat upstream Rhizomatic as the source to align with, not material to copy. Confirm license/permission before importing code, spec text, vectors, or tests.
